PUT YOUR B-NAZANIN WEB FONT HERE
================================

The site is already wired to use B Nazanin for Persian text.
To turn it on, add a licensed B-Nazanin web font file to THIS folder,
named exactly:

    BNazanin.woff2     (best — smallest, fastest)

Optionally also add, as fallbacks for very old browsers:

    BNazanin.woff
    BNazanin.ttf

That's it — no code changes needed. Once the file is here and pushed to
GitHub, Persian text uses B Nazanin automatically.

UNTIL you add the file, Persian text uses "Noto Naskh Arabic" (a free,
high-quality Google web font) so the site always looks correct.

NOTE ON LICENSING: B Nazanin is a commercial font. Please make sure you
have the right to use it as a web font (embedding it in a website) before
publishing. If you don't, the built-in Noto Naskh Arabic fallback is free
to use, or consider "Vazirmatn" (already loaded, open-source).

To convert a .ttf to .woff2, you can use a free online converter such as
the one at fonts.google.com tools, or transfonter.org.
